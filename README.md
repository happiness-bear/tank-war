# 坦克大战

**由于能力有限，我没有完全完成的项目，接下来介绍一下时该项目的功能、代码实现以及显示**

- ## 项目功能

  玩家需操控我方坦克去消灭敌方坦克，消灭两辆即为获胜，游戏结束

  | 按键 |   功能   |
  | :--: | :------: |
  |  A   | 向左移动 |
  |  D   | 向右移动 |
  |  W   | 向上移动 |
  |  S   | 向下移动 |
  | 空格 | 发射子弹 |
  |  Q   | 退出游戏 |

- ## 代码实现

  

  |  模块  |             功能             |
  | :----: | :--------------------------: |
  |  Wall  | 设置墙体，包括砖块、铁块等。 |
  |  Home  |          设置大本营          |
  | Bullet |           子弹设置           |
  |  Tank  |    设置我方坦克与敌方坦克    |
  |  Game  |        游戏资源初始化        |
